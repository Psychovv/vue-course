# Cookin' Up

Projeto desenvolvido durante um curso da Alura com foco em Vue.js e desenvolvimento front-end.

## Sobre o projeto

O Cookin' Up é uma aplicação de receitas criada para praticar conceitos de componentização, estilização e organização de projetos utilizando Vue.js.

## Tecnologias utilizadas

- Vue.js
- Vite
- HTML5
- CSS3
- JavaScript
